# Pomodoro Timer

A minimalistic and modern Pomodoro timer built with Next.js.

## Features

- Start, pause, and reset the timer
- Adjustable work/break time
- Audio alert when time is up

## Tech Stack

- Next.js, TypeScript

## Installation

```bash
git clone https://github.com/bm78g/pomodoro-timer.git
cd pomodoro-timer
npm install
npm run dev
```
